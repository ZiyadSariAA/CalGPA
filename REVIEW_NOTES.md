# CalGPA — App Review Notes

## What is CalGPA?
CalGPA is a GPA calculator app for university students in Saudi Arabia. It supports both the 4.0 and 5.0 grading systems used by Saudi universities.

## Key Features
- **GPA Calculator** — Calculate semester and cumulative GPA
- **Dashboard** — Track GPA progress across semesters
- **CV Builder** — Create professional resumes with optional AI assistance
- **Opportunities** — Browse COOP/training programs

## Permissions & Data
- **No login required** — The app works fully offline with local storage
- **No camera, location, or contacts access**
- **No in-app purchases or subscriptions**
- **Firebase Firestore** is used only for read-only app configuration and notifications

## AI Feature
- The CV builder includes an optional AI writing assistant (powered by a cloud function)
- It helps improve resume bullet points and summaries
- Usage is rate-limited per day
- No user data is stored on the server

## Language
- The app is in **Arabic** with a right-to-left (RTL) layout
- All UI text is in Arabic

## Demo Credentials
- No login or account needed — the app is fully accessible without authentication
